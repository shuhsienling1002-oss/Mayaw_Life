\# 🏗️ Life Architect: 人生建築師 v1.0



這是一個基於 \*\*FP-CRF (First-Principle Collaborative Reasoning Framework)\*\* 邏輯設計的財務模擬工具。專為建築工程從業人員設計，用於模擬不同理財策略在 10~20 年後的資產差距。



\## 核心功能

\- \*\*多重模式模擬\*\*：比較「隨性消費」、「強制儲蓄 (影子房東)」與「技能投資」的長尾效應。

\- \*\*視覺化反饋\*\*：使用 Streamlit 進行即時參數調整與圖表繪製。

\- \*\*物理必然性推導\*\*：計算複利效應與技能折舊的對沖關係。
